# Personal Website

Ini adalah repositori untuk website portofolio pribadi saya. Website ini dibangun murni menggunakan HTML, CSS, dan JavaScript dengan desain yang modern, interaktif, dan responsif.

## 🚀 Fitur Utama

- **Desain Modern & Gelap (Dark Theme):** Menggunakan palet warna Dark Navy, Ocean Blue, dan efek *Glassmorphism* pada navigasi.
- **Responsif (Mobile-Friendly):** Tampilan otomatis menyesuaikan dengan baik di berbagai ukuran layar (Desktop, Tablet, dan Mobile).
- **Animasi Dinamis:** Dilengkapi efek pengetikan (*Typing Effect*) menggunakan [Typed.js](https://github.com/mattboldt/typed.js/) dan efek elemen muncul saat *scroll* (*Scroll Reveal*) menggunakan [Anime.js](https://animejs.com/).
- **Navigasi Single-Page:** Pengalaman gulir (*scroll*) yang mulus antar bagian (About, Experience, Skills, dan Projects).

## 🛠️ Teknologi yang Digunakan

- **HTML5** untuk struktur halaman semantik.
- **CSS3** untuk desain UI, tata letak (Flexbox & Grid), dan efek transisi.
- **Vanilla JavaScript** untuk interaktivitas fungsional (menu hamburger, transisi navbar).
- **Font Awesome** untuk kebutuhan ikon sosial media dan UI.
- **Google Fonts (Outfit)** untuk tipografi modern.

## 📂 Struktur Direktori

## 💻 Cara Menjalankan di Komputer Lokal

1. **Clone repositori ini:**
   ```bash
   git clone https://github.com/codeyannn/PersonalWebsiteMuhammadMaulanaAdrian.git
   ```
2. **Masuk ke direktori:**
   ```bash
   cd PersonalWebsiteMuhammadMaulanaAdrian
   ```
3. **Buka file:**
   Klik ganda pada file `index.html` untuk membukanya di browser pilihan Anda (Google Chrome, Firefox, dll). Anda juga bisa menggunakan ekstensi *Live Server* di VS Code jika ingin melakukan modifikasi secara langsung.
