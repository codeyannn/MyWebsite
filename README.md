# Personal Website

This is the repository for my personal portfolio website. It is built entirely using HTML, CSS, and JavaScript with a modern, interactive, and responsive design.

## 🚀 Key Features

- **Modern & Dark Theme:** Utilizes a color palette of Dark Navy, Ocean Blue, and *Glassmorphism* effects on the navigation bar.
- **Responsive (Mobile-Friendly):** The layout automatically adapts beautifully across various screen sizes (Desktop, Tablet, and Mobile).
- **Dynamic Animations:** Features a *Typing Effect* using [Typed.js](https://github.com/mattboldt/typed.js/) and *Scroll Reveal* animations using [Anime.js](https://animejs.com/).
- **Single-Page Navigation:** Smooth scrolling experience across sections (About, Experience, Skills, and Projects).

## 🛠️ Technologies Used

- **HTML5** for semantic page structure.
- **CSS3** for UI design, layout (Flexbox & Grid), and transition effects.
- **Vanilla JavaScript** for functional interactivity (hamburger menu, navbar transitions).
- **Font Awesome** for social media and UI icons.
- **Google Fonts (Outfit)** for modern typography.

## 💻 How to Run Locally

1. **Clone this repository:**
   ```bash
   git clone https://github.com/codeyannn/PersonalWebsiteMuhammadMaulanaAdrian.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd PersonalWebsiteMuhammadMaulanaAdrian
   ```
3. **Open the file:**
   Double-click on the `index.html` file to open it in your preferred browser (Google Chrome, Firefox, etc.). You can also use the *Live Server* extension in VS Code if you want to make live modifications.
